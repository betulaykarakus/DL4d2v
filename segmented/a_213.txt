空腹血糖 FPG 正常值 为 3.6.mmol ／ L
空腹血糖 7.mmol ／ L 或餐 后 小时 血糖 P2hPG 11.mmol ／ L 时 可 诊断 为 糖尿病

尿糖 U GLU )
正常 情况 下 尿液 只 含有 微量 葡萄糖 尿糖 定性 检查 呈 阴性
当 血糖浓度 增高 到 一定 程度 8.10.mmol ／ L 时 肾小管 将 不能 全部 吸收 回 尿液 葡萄糖 尿糖 增高 呈 阳性 临床用 ＋ 号 表示
一般 情况 下 尿糖 可以 反映 出 血糖 情况 但 尿糖 还 受 其他 因素 影响 因此 有时 与 血糖 并 不 完全一致
例如 当 患者 有 肾 小 动脉硬化 肾脏 疾病 时 由于 肾糖阈 增高 患者 尽管 血糖 很高 尿糖 却 往往 阴性
再 如 妊娠期 妇女 肾糖阈 往往 减低 尽管 血糖 不高 尿糖 可 呈 阳性
因此 尿糖 结果 仅供参考 而 不能 作为 诊断 依据
糖化血红蛋白 HbA1c )
由于 血糖 受 饮食 运动 药物 影响 而 波动 因此 测定 一次 血糖 只能 反映 取血 当时 血糖水平 不能 反映 采血 前 一段时间 血糖 情况 全貌
而 糖化血红蛋白 HbA1c 是 血红蛋白 与 葡萄糖 非酶促 结合 产物 合成 速度 与 红细胞 所 处 环境 糖 浓度 成正比
由于 红细胞 寿命 是 12天 半衰期 是 6天 故 糖化血红蛋白 HbA1c 可以 反映 采血 前 个 月 平均 血糖水平 正常值 为 ％
． 糖化 血清蛋白 GSP 果糖 胺法 )
糖化 血清蛋白 GSP 是 葡萄糖 与 血清蛋白 非酶促 结合 产物 血清蛋白 半衰期 短 因此 糖化 血清蛋白 GSP 仅 反映 检查 前 周 血糖 总体 控制 状况 正常值 应为 1.2.mmol ／ L

C 肽 是 内源性 胰岛素 最后 生成 时 分子 离解 产物 正常人 空腹 血浆 C 肽 值 为 0.4.g ／ L

． 尿 微量 白蛋白 MALB 测定
