)
怀孕 之前 体重 超重 即 体重 超过 理想 体重 2以上 )
)
属于 糖尿病 高发 种族 一员
这些 种族 包括 西班牙人 种 黑 皮肤 人种 美洲 土著人 种 和 亚洲 人种
)
尿含有 葡萄糖
)
糖耐量受损 血糖水平 高于 正常 但是 尚未 达到 诊断 糖尿病 标准 )
)
有 糖尿病 家族史 即 父母 或者 同胞 患有 糖尿病 )
)
曾经 分娩 过 体重 大于 磅 约 4.0公斤 婴儿
)
曾经 患有 妊娠期 糖尿病
)
羊水 过多
但是 很多 患有 妊娠期 糖尿病 妈妈 并 没有 明确 危险 因素 所以 如果 不 符合 上面 所 列出 情形 不要 大意 最 重要 是 定期 进行 产前检查
为检查 出 妊娠期 糖尿病 妈妈 们 需要 进行 一项 叫做 口服葡萄糖耐量试验 检查
即将 7克 无水 葡萄糖 或者 82.克 含有 一 分子 水 葡萄糖 融于 2530毫升 水分钟 内 饮 完 两 小时 内后 再测 静脉 血浆 糖量
血糖水平 大于 14毫克 分升 就 可以 认为 是 异常 但是 并 不能 根据 该 结果 诊断 为 糖尿病
为确诊 糖尿病 还 需要 在 几个 小时 后 测量 血糖水平
对于 那些 具有 高危因素 孕妇 来说 一次 正常 糖耐量 试验 结果 并 不能 说明 问题 还 需要 在 妊娠 第 22周时 连续 测量 才能 排除 妊娠期 糖尿病 诊断
如果 不幸 站 到 妊娠期 糖尿病 妈妈 行列 的话 不要 害怕 悲伤 有 很好 方法 可以 控制 糖尿病
下面 就是 一些 控制 妊娠期 糖尿病 方法
)
一天 四次 检测 血糖水平 早餐 前 以及 餐后 小时
监测 每餐 前 血糖水平 是 很 有 必要 )
)
监测 尿 酮体 水平 酮体 是 脂肪酸 代谢 产物 总称 其 水平 升高 提示 糖尿病 控制 不佳 )
)
遵循 健康 糖尿病 饮食
通常 需要 合理 分配 每餐 饭 进餐量
)
在 获医生 许可 后 适量 运动
)
监测 体重 预防 体重 不正常 增加
)
如果 有 必要 的话 注射 胰岛素
