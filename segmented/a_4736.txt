型 糖尿病
如何 使用 胰岛素
研究 发现 每天 一次 胰岛素 是 一个 推荐 起始 剂量
你 患有 型 糖尿病 并且 需要 开始 使用 胰岛素 吗
科学家 们 已经 为 你 研究 出 最 恰当 胰岛素 起始 用法
一项 最新 研究 表明 每天 睡前 使用 一次 胰岛素 是 你 应该 踏出 第一步 这样 做 比 其他 胰岛素 剂量 策略 要 安全
但 这项 发现 还 只是 初步 研究 结果 所以 糖尿病 病人 还是 应该 先 咨询 自己 医生 看看 他们 意见
糖尿病 专家 们 对 以下几点 已经 非常 肯定
把 你 糖化血红蛋白 A1c
保持 在 以下 可以 减少 你 心脏病 中风 和 其他 糖尿病并发症 机率 而 如果 你 需要 使用 胰岛素 才能 达到 糖化血红蛋白 控制目标 那么 你 应该 开始 使用 胰岛素
什么 是 糖化血红蛋白 A1c
你 体内 糖化血红蛋白 A1c 数量 能够 反映 你 过去 到 个 月 血糖控制 水平
糖化血红蛋白 A1c 控制目标 应该 在 以下
如果 型 糖尿病 病人 靠 健康 生活方式 和 口服药物 不能 达到 这个 目标 的话 他们 应该 开始 使用 胰岛素
这个 新 研究 以 70个 来自 英国 和 爱尔兰 型 糖尿病 病人 为 研究 对象
这些 病人 过去 口服药物 剂量 已经 达到 最大 剂量 这些 药物 包括 二甲双胍 和 其他 糖尿病 药如 磺脲 类 包括 达安 疗 优降糖 泌 乐）
尽管 摄入 最 大量 以上 药物 病人 糖化血红蛋白 A1c 水平 还是 太高 介于 到 1之间
型 糖尿病人 与 胰岛素
所有 研究 对象 都 开始 使用 胰岛素 但 他们 采用 以下 种 用药 方法 其一种
每天 次
一天 次 餐前 用
每天 睡前 一次
在 使用 胰岛素 一年 以后 其一些 病人 达到 他们 糖化血红蛋白 A1c 控制目标
在 每天 使用 次 胰岛素 病人 中有 1达到目标 餐前 用 胰岛素 病人 有 四分之一 达到目标 而 每天 睡前 使用 次 胰岛素 病人 有 达到目标
每天 次 胰岛素
避免 危险 低血糖 和 体重增加 之间 有 一条 紧密 纽带 联系
每天 睡前 使用 一次 胰岛素 病人 体重增加 机率 最低 在 使用 胰岛素 首年 发生 低血糖 现象 机率 最低
