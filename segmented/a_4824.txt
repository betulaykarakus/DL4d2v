有 调查 显示 截至 去年 我国 糖尿病患者 达到 1.1亿 全世界 每 个 糖尿病患者 就 有 一个 来自 中国
可见 控糖 已成 一件 刻不容缓 事情 而 在 控糖 过程 运动治疗 是 十分 重要 手段
糖尿病人 运动 有利于 增加 肌肉组织 对 葡萄糖 利用 达到 降低血糖 目的
但 糖尿病患者 在 运动 时候 应该 遵循 哪些 原则 呢
今天 我 就 来 给 大家 讲讲 糖尿病患者 在 运动 中应 注意 事项
运动前 注意事项
首先 应 根据 个人 情况 决定 运动 方式 时间 和 运动量
糖尿病患者 运动 方案 在 实施 之前 最好 到 医院 进行 一次 全面 系统 检查 包括 血压 血糖 糖化血红蛋白 心电图 眼底 肾功能 检查
有时 心功能 检查 有 必要
患者 要 与 医生 建立 良好 关系 请 医生 制订 合理 运动 计划
所有 接受 运动治疗 糖尿病人 都 要 进行 全面 病史 询问 和 检查
如果 是 老年人 或者 肺病 患者 有 条件 应加测 肺 功能 测血压 查 眼底 有 严重 心肺功能 障碍 或 眼底 有 出血 病史 则 不应 参加 运动 如 运动后 血压 上升 眼底 出血 或 心肌缺血 应 停止 或 减少 运动
运动时 注意事项
正式 运动前 要 做好 11分钟 热身运动 如 伸腰 抬腿 慢走 使 肌肉 活动 起来 避免 肌肉 拉伤
运动 开始 后要 由 慢到 快 运动 强度 由小到大 1分钟 逐渐 加量 每次 运动 持续 时间 一般 在 23分钟 为宜
要 选择 合适 鞋袜 要 特别 注意 鞋袜 密闭性 和 通气 性 由于 不少 糖尿病患者 合并 有 周围神经病变 致使 足部 感觉 减退 或 足底 皮肤 增厚 变硬 足部 容易 受损 锻炼 时宜 穿 宽松 鞋底 柔软 舒适 鞋面 透气 好 鞋 如 运动鞋 胶鞋
应 选择 安全 运动场 寻找 合得来 运动 伙伴 避免 单独 一人 运动
如果 是 在 炎热 夏季 穿 轻便 棉织品 它 能 吸收 汗水 并 使 潮气 蒸发 从而 保持 正常 体温 还应 注意 在 运动 和 运动后 及时 补水 以免 脱水
如果 是 在 冬季 不要 穿 一层 厚 衣服 最好 穿 多层 衣服 这样 既保证 保暖性 最主要 是 在 运动 过程 如果 觉身体 发热 可以 适当 脱掉 一两件 衣服 如果 穿 一件 厚 衣服 运动 脱掉 很 容易 着凉 感冒 此外 还要 戴帽子 和 手套 来 保暖
年龄 大 糖尿病患者 或者 以前 运动时 受过伤 运动 者 还要 注意 对 个别 易 受伤 部位 保护 最好 戴 护具 如 护膝 护腕 等
结束 时 不要 突然 停止 应 逐渐 放慢 节奏 做 1分钟 左右 恢复 活动 再 坐下 来 休息
必须 注意 逐渐 增加 运动量 和 强度 循序渐进 切不可 操之过急 应 随着 体质 增强 和 身体 适应 程度 不断 增加 运动量 做到 持之以恒 每周 保证 至少 次 运动
为 防止 发生意外 运动时 要 随身携带 记录 有 本人 姓名 年龄 家庭 住址 及 联系电话 糖尿病病情 卡
注意 心率 变化 及 自我感觉 如 感觉 身体状况 不好 应立即 停止 运动 并 找 他人 救助
随身携带 几块 糖果 以 防备 发生 低血糖 时 及时 补充 糖分
运动 结束 后 应立即 更换 汗湿 衣服 以防 感冒
天气炎热 夏季 应及时 补充 水分 但 不能 一次性 过 多饮水 易感 疲劳 增加 胃 负担 ）
做好 运动 记录 应 定期 监测 运动前 运动后 和 运动 血糖值 变化
如 有 不适 请 医生 或 专业 护士 进行 运动 处方 相应 调整
此外 糖尿病人 在 运动 过程 经常出现 低血糖症 和 足 损伤 这时 应该 注意 以下几点
不要 在 清晨 空腹 时 剧烈运动 因为 清晨 是 人体 一天 血糖 最低 时间 此时 运动 特别 是 运动量 大时 容易 发生 低血糖 反应 提倡 餐后 运动
避免 在 使用 胰岛素 或 口服降糖药物 后 未能 及时 进食 时 运动
合理安排 运动量 不要 超 大量 剧烈运动
运动时 要 携带 易于 吸收 碳水化合物 如 葡萄糖 片 软饮料 葡萄干 糖果 等
运动 类型 如何 选择
要 根据 自己 条件 和 周围 环境 因地制宜 选择 运动 项目 尽量 选择 安全 便利 运动 形式 如下 运动 可 供选择
轻度 运动
购物 散步 广播操 太极拳 气功
中度 运动
快走 慢跑 骑车 上下 楼梯 健身操
强度 运动
快 跑 跳绳 爬山 游泳 球类
要 结合 本人 病情 轻重 是否 伴有 合并症 选择 运动 方式
对 血糖 不稳定 波动 较大 者 一般 不 主张 持续时间长 和 运动量 大 运动
伴有 心脏病 者 不 主张 选择 中等量 以上 运动
伴有 肥胖 和 膝关节 疾病 者 不 主张 进行 中强度 以上 需要 下肢 承重 运动
运动 时间 和 强度 如何 安排
糖尿病人 以餐 后 运动 较 适宜 这样 有利于 降低 餐后 高血糖 早餐 后 是 一天 血糖 最高 时刻 故 安排 在 早餐 后 小时 运动 最佳 从 第一口 饭算起 而 对于 早餐 前 和 晚 餐后血糖 控制 不好 患者 可以 选择 早餐 前 和 晚餐 后 时段 运动
每天 坚持 运动 益处 最大 每周 不能 少于 次 每次 需 坚持 36分钟
适宜 运动量 是
运动时 感觉 全身 发热 微汗 轻度 肌肉 酸痛 次日 感觉 精神 充沛 有 运动 欲望 食欲 和 睡眠 良好
运动 量过大 表现 有
大汗淋漓 气喘 胸闷 不思 饮食 次日 身体 乏力 精神 不佳 需要 及时 减少 运动量
运动时 心率 是 判断 运动 强度 重要 指标
比较 科学 判断 方法 是 最大 安全 运动 心率 17年龄 来 确定 你 运动量 大小
比如 你 年龄 是 6岁 那么 你 运动 强度 即 你 运动时 心率 应该 是 17610次 分

若 运动后 身体 无 发热 无汗 脉搏 无 明显 变化 或 有 变化 运动 停止 后 分钟 内 迅速 恢复正常 表明 运动量 过 小
