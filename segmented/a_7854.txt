糖尿病 饮食控制 要求 遵循 两个 原则
第一个 原则 是 一个 平衡 三个 兼顾 即 平衡 饮食 兼顾 控制血糖 血脂 血压 体重 兼顾 并发症 防治 和 兼顾 个人 生活习惯 和 饮食 爱好
第二 平原 则 是 避轻 就 重 根据 糖尿病 病情 和 并发症 有无 和 程度 优先 控制 或 解决 主要 问题 例如 糖尿病患者 并发 糖尿病肾病 应该 首先 考虑 控制 糖尿病肾病 在 可能 情况 下 兼顾 解决 其他 问题 如 高血脂 )
糖尿病 饮食控制 关键 是 合理 选择 食物
一般来说 糖尿病患者 每天 饮食 应当 有 以下 食物 构成 即 主食 粮食 奶类 蔬菜 水果 肉 蛋 鱼 虾 豆类 及 豆制品 以及 植物油 等
主食 选择
每日 三餐 必须 要 有 主食 有人 认为 主食 含糖 多 吃升高 血糖 不吃 主食 血糖 当然 就 不 升高 ”
诚然 粮食 含有 糖 食用 粮食 是 要 升高 血糖 但 如果 主食 品种 和 数量 选择 得当 餐后血糖 不会 急剧 升高
另一方面 由于 主食 吃 少 就 有 可能 摄入 更多 含 脂肪 和 蛋白质 较高 食物 会 加重 已经 存在 并发症
一般来说 轻体力 活动 糖尿病患者 每天 可食用 主食 1530克
可 选择 一部分 精米 白面 并 搭配 粗粮 如 荞麦 燕麦 高粱 鲜玉 玉米面 )
奶类 选择
可 选择 牛奶 鲜奶 为宜 奶粉 和 酸奶 以未 加糖 调味 酸奶 果料 酸奶 为宜 )
不宜 选择 是 奶酪 奶油 炼乳 因 其 脂肪 含量 较高
不宜 选择 全脂 加糖 奶粉 和 加糖 酸奶
如果 存在 高脂血症 可 选择 脱脂奶 或 脱脂奶粉
推荐 每天 摄入量
鲜奶 25毫升 左右 或 奶粉 11克 或 酸奶
1520毫升
蔬菜 选择
蔬菜 对 血糖 影响 较小 淀粉类 蔬菜 除外 因 蔬菜 含 丰富 无机盐 维生素 和 纤维素
另外 洋葱 甘蓝 西红柿 还 含有 生物 类黄酮 其为 天然 抗氧化剂 能 维持 微血管 正常 功能 等
因此 提倡 糖尿病患者 多 食用 蔬菜
一般来说 推荐 每天 至少 摄入 50克 新鲜 蔬菜 而且 品种 要 多一些 至少 种 最好 种
水果 选择
水果 含有 丰富 维生素 无机盐 和 膳食纤维 有益于 健康
但是 水果 含有 碳水化合物 如 葡萄糖 这些 糖类 消化 吸收 较 快 升高 血糖 作用 比 复合 碳水化合物
如 粮食 要 快 所以 对 血糖控制 不 理想 糖尿病患者 来说 最好 不要 食用 水果
空腹 和 餐后血糖 分别 在 7.mmol L 和 8.0mmol L 以下 并 稳定 周 以后 才 可食用
食用 水果 前后 最好 要 监测血糖 根据 血糖 情况 及时 调整 选择 合适 水果 品种
一般来说 糖尿病患者 一天 可以 食用 水果 1520克
在 选择 水果 时 原则上 优先选择 含糖 较低 水果 例如 梨 桃 草莓 樱桃 柚 李子 猕猴桃 等
食用 水果 时间 宜 安排 在 两顿 正餐 之间 不要 在 餐后 马上 食用
肉 蛋 鱼 虾 选择
可 选择 新鲜 瘦猪肉 瘦 牛肉 瘦羊 肉
禽肉 或 其 制品
尽量少 食 或 不食 动物内脏 皮肤 这些 胆固醇 含量 较高 食物
蛋类 可 选择 鸡蛋 鸭蛋 鹅蛋 鸽蛋 等
对 糖尿病患者 采说 如 选择 鸡蛋 天 个 鸡蛋 即可 如果 并发 高胆固醇血症 可 隔天 个 全蛋 之间 可食用 蛋清 不用 蛋黄
对 合并 高血压 糖尿病患者 来说 不宜 食用 皮蛋 咸蛋
鱼虾 脂肪 多 由 不饱和脂肪酸 组成 特别 是 n 长链 多不饱和脂肪酸 具有 降低 血脂 防止 动脉粥样硬化 作用
鱼子 虾子 胆固醇 含量 较高 合并 高胆固醇血症 患者 不宜 选择 )
一般来说 糖尿病患者 每天 可食用 肉蛋 鱼虾 共 10～
20克

大豆 及其 制品 是 糖尿病患者 理想 食物 之一
糖尿病患者 可 选择 豆腐 豆浆 豆腐干 豆腐 丝 豆腐脑 腐竹 千张 素鸡 但 不宜 选择 油豆腐
推荐 每天 食用量 为 豆腐 1520克 或 豆腐干 10克 或 豆腐 丝 510克 或 豆浆 2550毫升
植物油 选择
糖尿病患者 不宜 选择 动物 油脂 作为 烹调油 而宜 选择 植物油
可 选择 植物油 有 橄榄油 茶子 油 豆油 花生油 葵花子油 等
其以 橄榄油 和 茶子 油为 优 但是 价格不菲
一些 植物油 如 椰子油 含 饱和脂肪 较高 糖尿病患者 不 适宜 食用
植物油 摄入量 一般 为 每天 23毫升 相当于 汤勺 )
