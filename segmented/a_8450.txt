糖尿病 号称 不死 癌症 ”
数据 显示 中国 成年人 糖尿病 患病率 为 11.处于 糖尿病前期 患者 占 总人口 50.%
可见 糖尿病 现状 十分 严峻
治疗 糖尿病 除要 注意 饮食 加强 运动 以外 药物 治疗 十分 关键
选择 合适 自己 药物 能够 更 有效 帮助 控制 病情
下面 就 带 大家 了解 一下 糖友 该 如何 选择 适合 自己 药物
糖友 选择 药物 大 标准
1
根据 糖尿病 类型 选择 药物
糖尿病 分为 型 糖尿病 和 型 糖尿病
型 糖尿病患者 必须 终生 使用 胰岛素
而 型 糖尿病患者 应 考虑 使用 口服 抗 糖尿病 药物 可 根据 自身 身体状况 选择 是否 使用 胰岛素 治疗 如果 型 糖尿病患者 在 在 饮食控制 运动 调理 以及 口服 抗 糖尿病 药物 效果 都 不好 并且 出现 急性 合并 视网膜病变 尿毒症 应激状态 并发症 手术 期 或 孕产 期 就 必须 使用 胰岛素 治疗
2
根据 患者 年龄 大小 选择 药物
年龄 不同 选择 药物 标准 会 不 一样
对 儿童 来说 型 糖尿病 使用 胰岛素 治疗 对于 型 糖尿病 目前 只有 二甲双胍 被 FDA 批准 用于 儿童 糖尿病 治疗
对 老年 患者 来说 因为 对 低血糖 耐受 能力差 不宜 选用 长效 强力 降糖药物 而应 选择 服用 方便 降糖 效果 温和 降糖药物
3
根据 血糖波动 情况 选择 药物
血糖 是 糖尿病 治疗 关键 指标
对于 单纯 餐后血糖 高 而 空腹 和 餐前 血糖 都 不 高 糖友 应 选择 糖苷酶抑制剂 治疗
对于 以 餐后血糖 升高 为主 伴有 餐前 血糖 轻度 升高 糖友 应选 用 苯甲酸 衍生物
而 对于 空腹血糖 餐前 小时 血糖高 不管 是否 有 餐后血糖 高 糖友 都 应选 用 磺脲 类 双胍类 或 噻唑烷二酮类 药物
4
根据 患者 体形 选择 药物
标准 体重 计算 方式 是 身高 c10如果 实际 体重 超过 理想 体重 1就 可 认为 是 偏胖 应选 用 双胍类 或 糖苷酶抑制剂 因为 这 类药物 有 胃肠道 反应 和 体重 下降 副作用 可以 帮助 超重 或 肥胖 糖友 减轻 体重
如果 实际 体重 低于 理想 体重 1则 可 认为 是 偏瘦 应 使用 胰岛素 促 分泌 剂 因为 这 类药物 有 增加 体重 副作用 适合 偏瘦 糖友 使用
5
根据 有无 其他 疾病 选择 药物
对于 同时 患有 高血脂 高血压 糖友 应 选择 使用 双胍类 噻唑烷二酮类 和 糖苷酶抑制剂
而 伴有 慢性 支气管炎 肺气肿 疾病 患者 要 慎用 双胍类药物
伴有 胃肠道 疾病 糖友 就 不要 使用 双胍类 和 糖苷酶抑制剂
患有 肝病 糖友 就要 慎用 噻唑烷二酮类 而 又 心 肝肾 肺 全身性 疾病 糖友 最好 使用 胰岛素 治疗
维持 血糖 稳定 是 治疗 糖尿病 长期 目标 药物 治疗 是 关键
但是 部分 糖友 血糖 刚刚 稳定 就 减药
那么 到底 糖尿病人 血糖 降 能 不能 减药 呢
血糖 降 能 不能 停药 或 减药
回答 是 否定 的
对于 多数 人 来说 血糖控制 良好 是 饮食治疗 运动治疗 和 药物 治疗 共同 作用 结果 口服 降糖药 作用 不容 低估 一旦 停 降糖药 很 有 可能 导致 血糖 再次 升高
自行 减药 或 停药 往往 就是 部分 糖尿病患者 长时间 血糖控制 不 理想 一个 重要 原因
建议 糖尿病患者 可以 再 血糖水平 降至 正常 一段时间 后 再 考虑 减药
即使 可以 减药 要 遵照 医嘱 循序渐进 不能 突然 停药 否则 会 诱发 意外 或 使 病情 加重
