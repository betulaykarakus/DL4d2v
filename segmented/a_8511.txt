医院 药房 有 很 多种 降糖药 这么 多种 降糖药 患者 应该 如何 选择 呢
专家 说 不同 厂家 降糖药 有 上 百种 但 总结 一下 降糖药 共分 大类
第一 大类 是 磺脲 类 代表 药物 有 优降糖 达美康
该类 药 可以 促进 胰岛素 分泌 帮助 人体 降血糖
因为 磺脲 类药物 有 增重 作用 所以 适合 体型 较瘦 糖尿病患者
第二 大类 是 双胍类 代表 药物 是 二甲双胍 片 等
该类 药 特点 是 刺激 肌肉组织 摄取 葡萄糖 以 增加 人体 对 血糖 消耗
双胍类药物 有降 体重 作用 适合 体型 较 胖 糖尿病患者 使用
第三 大类 是 苷 酶类 代表 药物 是 拜糖平 等
它 主要 特点 是 抑制 肠道 对 糖 吸收 人体 吸收 糖少 血糖 自然 就会 下降
餐后血糖 高 患者 优先选择 苷 酶类 药物
第四 大类 是 胰岛素增敏剂 代表 药物 是 吡格 列酮片
二型糖尿病 特点 是 人体 对 胰岛素 吸收 利用 功能 减弱 该药 可以 增加 人体 对 胰岛素 利用
第 五大类 药是 DPP抑制剂 代表 药物 是 西格 列丁片
它 原理 是 通过 促进 胰岛素分泌 同时 延长 胃 排空 时间 抵制 食欲
专家 同时 强调 选择 中西医 结合 治疗 糖尿病 可以 达到 更佳 治疗 效果 特别 是 出现 糖尿病肾病 并发症 后 更要 中西医 结合 治疗 中医 在 消除 蛋白尿 保护 肾功能 方面 有着 独特 优势 这点 连 西医 专家 是 认同
