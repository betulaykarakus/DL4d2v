血糖 顾名思义 是 血液 糖
糖 是 我们 身体 不可 缺少 营养物质 之一 人 摄入 食物 以后 经过 消化系统 转化 为 单糖 如 葡萄糖 进入 血液
血糖 其实 就是指 血液 葡萄糖
血糖 通过 血液 被 运送 到 全身 各个 组织细胞 分解 燃烧 产生 人体 所 需 能量
血糖 随 血液 流经 全身 与 全身 组织细胞 代谢 有 密切关系 因此 血糖 稳定 与否 影响 到 身体 正常 生理活动 机能
正常 情况 下 血糖 处于 一种 动态平衡 状态 每日 消耗 与 补充 同时 进行
而 糖尿病患者 血糖 则 是 失衡 的
人体 激素 负责 调节 血糖
人 体内 有 多种 激素 它们 共同 组成 一个 糖代谢 调节 系统 维持 着 血糖 动态平衡
这些 激素 有 胰岛素 胰高血糖素 肾上腺素 糖皮质激素 生长激素 和 甲状腺素
其胰岛素 是 体内 唯一 可以 降低血糖 激素 它 主导 着 葡萄糖 在 体内 合成 和 转化 是 调节 血糖 最 重要 激素
其余 五种 激素 作用 主要 是 升高 血糖
正常 情况 下 升高 血糖 激素 与 胰岛素 处在 一个 平衡状态 从而 使血糖 保持平衡
血糖 对 胰岛素 分泌 有 制约 作用 当 血糖升高 时 胰岛 就会 接受命令 多 制造 胰岛素 降低血糖 而 血糖 过低 时 胰岛 会 减少 或 停止 制造 胰岛素 血糖 就 不再 下降 了
所以 健康 人 不论 进食 与否 一般 血糖 含量 都 比较稳定
糖尿病 有 好几种 类型 常见 有 一型 和 二型
一型糖尿病 是 胰岛素 分泌 少 以 中青年 居多
二型糖尿病 则 是 人体 分泌 胰岛素 并 不少 只是 人体 对 胰岛素 利用 度 下降 胰岛素 工作效率 低 了
临床 医生 常给 患者 抽血 查看 胰岛素 功能 和 C 肽 胰岛素 抗体 检查 来 判断 患者 是 一型 还是 二型糖尿病
如果 是 一型糖尿病 必须 打 胰岛素 如果 是 二型糖尿病 则 可以 吃药 控制
不过 有些 二型糖尿病患者 适合 打 胰岛素 比如 先有 心脑 肾 疾病 又 出现 糖尿病 或者 先有 糖尿病 又 出现 心脑 肾 并发症 或者 是 妊娠期 糖尿病患者 均 适合 打 胰岛素 因为 胰岛素 本身 是 人体 一种 激素 对 人体 无损伤 而 药物 则 对 肝肾 有 损害 所以 面对 有 并发症 或 妊娠期 糖尿病患者 还要 选择 无 不良反应 治疗 方法
