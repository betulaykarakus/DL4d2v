胶囊 换 片剂 血糖 突升
孙 老伯 从 老家 来 郑州 看望 女儿 在 女儿 一再 挽留 下 孙 老伯 就 多 住 些 日子
由于 孙 老伯 患有 型 糖尿病 一直 服用 格列 吡嗪 胶囊 控制血糖 可 不巧 是 他 带来 药 马上 就要 吃 完 于是 他 让 女儿 去 药店 再 买些
但 女儿 跑遍 周围 药店 怎么 找 不到 格列 吡嗪 胶囊 却 都 有 一种 类似 格列 吡嗪 片 售货员 介绍 说 虽然 两种 药物 一个 是 胶囊 一个 是 片剂 但 它们 药理 是 一样 的
于是 孙 老伯 女儿 就 买回去 让 父亲 按照 说明书 服用
可 半个 月 后 孙 老伯 到 医院 查 血糖 发现 血糖 很高 这是 怎么回事
他 把 改服 格列 吡嗪 片 事 说 给 医生 听 并 把 药品 说明书 拿给 医生 看
医生 这才 发现 孙 老伯 血糖升高 原因 就是 擅自 换药 引起 的
原来 孙 老伯 以往 服用 格列 吡嗪 胶囊 规格 是 每粒 5mg 医生 让 他 每次 吃 一粒 而格列 吡嗪 片 规格 是 每片 0.2mg 按照 说明书 孙 老伯 每次 服用 一片 这样 每天 服用 药量 减少 一半 所以 孙 老伯 血糖 就 升高 了
换药 需注意 单片 剂量
据 郑州市 第一 人民 医院 内分泌科 副主任医师 何 向前 介绍 糖尿病患者 在 更换 服用 药物 时 最好 在 医生 指导 下 进行
因为 不同 生产厂家 和 不同 类型 药物 一般 单个 规格 会 存在 差别 服药 剂量 和 时间 频率 相应 会 有 不同 要求 不 在 医生 指导 下 擅自 换药 一旦 药量 突然 减少 就会 导致 病情 加重 有时 甚至 还会 出现 急性并发症
相反 如果 服药 剂量 增大 还会 出现 血糖 过低 严重 会 使 患者 昏迷 甚至 出现 生命危险
因此 糖尿病人 换药 一定 要 在 医生 指导 下 进行
另据何 向前 介绍 治疗 糖尿病 药物 一般 分为 普通 片 控释 片 和 缓释片 种类 不同 服用 时间 有所 差异
而且 不同 药物 代谢 都 各有特点 有些 是 长效 有些 是 短效 有些 是 降 空腹血糖 有些 是 降 餐后血糖 有些 是 消除 或 减轻 胰岛素 抵抗 的
因此 不同 病症 适用 药物 会 有所不同 这 是 糖尿病患者 切勿 盲目 随意 更换 药物 原因
