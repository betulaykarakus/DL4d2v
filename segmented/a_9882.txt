由于 我国 目前 现状 即使 是 接受 胰岛素 糖尿病病友 仍 有 许多 没有 条件 在 家进行 病情 自我 监测 此时 到 医院 定期 检察 血糖 和 尿糖 就 显格外 重要 因为 医生 必需 据此 来 调整 其 胰岛素 使用 剂量
另外 每 一个 病友 都 应该 定期 到 医院 完成 下列 在 家不易 或 不能 进行 检察 这些 检察 同时 又 是 格外 必要 的
以下 是 糖尿病患者 须要 定期 到 医院 进行 检察 项目 及 实际意义
糖化血红蛋白 检察 HbA1C ）
糖化血红蛋白 检测 与 陈规 血糖监测 方法 结合 有助于 指导 糖尿病患者 和 医师 制定 合理 治疗 方案 从而 使 血糖 得到 较 好 操控 最终 减少 糖尿病 及其 并发症 发生 危急性
尿 酮体 检察
酮体 是 体内脂肪 代谢 中间 产物 正常人 尿酮体 含量 极微 定性 试验 为 阴性
尿酮体 过多 称酮尿症 可见 于 糖尿病 酮症酸中毒 妊娠 强烈 呕吐 子痫 强烈 运动 饥饿 应急 状态 等
胰岛素 或 C 肽 检察
C 肽 与 胰岛素 之间 有 稳定 比例 关系 且 不受 胰岛素 抗体 干扰 注射 外源性 胰岛素 不含 C 肽 所以 测定 血C 肽 水平 可以 反应 内生 胰岛素 水平 还 可 了解 细胞 功能
